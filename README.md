# MERN--SEARCH-ENGINE

A Google API fueled book search engine!

## User Story

```md
AS AN avid reader
I WANT to search for new books to read
SO THAT I can keep a list of books to purchase
```
## Technology Used

  * apollo-server-express
  * apollo/client
  * bcrypt
  * express
  * graphql
  * jsonwentoken
  * mongoose
  * nodemon
  * concurrently
